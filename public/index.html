URL Shortener Service

Stories:
  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example Usage:

https://pure-tor-87235.herokuapp.com/new/www.google.com

Example Output

{original_url: "www.google.com", short_url: https://pure-tor-87235.herokuapp.com/0a137b375c}

Usage:

https://pure-tor-87235.herokuapp.com/0a137b375c

Will redirect to www.google.com
